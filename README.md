<h2>How to start?</h2>

```bash
(in /backend)

$ python3 -m venv venv
$ . venv/bin/activate (for Minsik: .venv/Scripts/activate)
$ pip install -r requirements.txt
$ python manage.py runserver

(in /frontend/vite-project)
$ npm install
$ npm run dev
```
